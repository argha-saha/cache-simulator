# Cache Simulator
Cache simulation program written in C++

## Features
- Multi-level Support: Simulates L1, L2, L3 cache hierarchies  
- Associativity Options:
    - Direct-mapped
    - N-way set associative
    - Fully associative
- Write Policies:
    - Write-back
    - Write-through
- Allocation Policies:
    - Write-allocate
    - No-write-allocate
- Eviction Policies:
    - LRU
    - Random
    - FIFO
- Statistics tracking
